1 · Bond Price
P = Σ C/(1+y)^t + FV/(1+y)^n C = coupon per period y = all-in yield per period = (r_f + CS/10000) / f r_f = risk-free rate (slider) CS = credit spread in bps (slider) n = total periods (= T x f) f = coupon frequency
CFA Level 1, Fixed Income
2 · Macaulay Duration
D_mac = Σ [(t/f) x PV(CF_t)] / P PV(CF_t) = CF_t / (1 + y)^t t in periods; t/f converts to years
CFA Level 1, Fixed Income
3 · Modified Duration
D_mod = D_mac / (1 + y/f) y = all-in yield = r_f + CS/10000 Interpretation: dP/P ~ -D_mod x dy (1st order)
CFA Level 1, Fixed Income
4 · Convexity
C = Σ [t(t+1) x PV(CF_t)] / [P x (1+y)^2 x f^2] Second-order sensitivity of price to yield. Higher convexity: benefits from large moves.
CFA Level 1, Fixed Income
5 · Price Change Estimate
dP/P ~ -D_mod x dy + 1/2 x C x (dy)^2 Duration effect : -D_mod x dy Convexity adjust. : 1/2 x C x (dy)^2
CFA Level 1, Fixed Income
6 · DV01
DV01 = D_mod x P x 0.0001 Dollar value change per 1bp yield move. Also known as PVBP (Price Value of a Basis Point).
CFA Level 1, Fixed Income
